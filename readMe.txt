Projeto desafio para vaga na empresa ProBrain

Bem-vindo à ProBrain,

Se você está lendo esse documento é porque foi selecionado para realizar 
nosso teste para novos desenvolvedores front end. Antes de começar os projetos, 
se familiarize com os serviços que serão usados para facilitar nossa visualização e avaliação:


CodeSandbox : Essa ferramenta chamada CodeSandbox é um editor online 
que permite criar aplicações web desde o protótipo até o deploy - tudo do site!

Jsfiddle: O Jsfiddle é uma IDE para aplicações web que também permite o 
desenvolvimento através do browser e o compartilhamento através de um link. 
Você deverá criar uma conta no Jsfiddle e entregará seu trabalho através 
do link de compartilhamento gerado. URL: jsfiddle.net

Tarefa 1 - Aplicativo API.

Desenvolver uma aplicação em React utilizando o Code Sandbox
e consumindo uma das seguintes API 
https://swapi.dev/ : Star Wars
https://pokeapi.co/  : Pokémon

A ideia da aplicação é livre, sendo obrigatório apenas consumir 3 recursos da API escolhida.

O que será analisado
- Domínio em ReactJS (utilizando Hooks)
- Domínio quanto ao consumo de API's
- Clareza e padrões de código
- Domínio do uso estilos de componente (Uso do Sass)

Tarefa 2 - Números.

Você deverá criar uma aplicação simples, com 4 diferentes implementação do “Números” utilizando HTML, Sass e JavaScript, e a ferramenta Jsfiddle. O usuário deverá ter a opção de escolher entre as 4 implementações (versões do jogo) inserir um número de 1 - 1000. Sua aplicação, então, deverá exibir o resultado de todas as implementações para o usuário. Para essa tarefa você deverá utilizar pelo menos dois dos princípios SOLID.

Implementação 1 - Se o número é par ou ímpar.
Implementação 2 - Se o número é primo ou não é primo.
Implementação 3 - Mostrar se o número existe na sequência de Fibonacci.
Implementação 4 - Mostrar a sequência de números onde múltiplos de 4 utilizem a palavra “pin”.
Após concluir as duas tarefas, envie os links de compartilhamento para vagas@probrain.com.br junto com quaisquer observações que queira fazer sobre seu trabalho. 

Obrigado pelo esforço, e boa sorte!
